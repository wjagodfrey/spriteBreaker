About

SpriteBreaker is an open source browser based sprite management tool build by Wilfred Godfrey. SpriteBreaker aims to make managing and updating spritesheets as easy as possible.

Quick start guide

  1. Load a fresh spritesheet using the  New button
  2. Add and name a new sprite eg "mario"
  3. Add and name a new action eg "stand"
  4. Set your view area
    • Scroll and click/drag on the small canvas to set the view area
  5. Add a new frame ( f on the keyboard)
  6. Set frame coordinates
    • Left click/drag on the large canvas to set the top left frame coordinate
    • Right click/drag on the large canvas to set the bottom right frame coordinate
  7. Save your spritesheet with the  Save button
  8. Copy the resulting JSON from the  Output area

Advanced

  • You can change the output fomat in the options dialog (the button)
  • Load and delete saved spritesheets with the  Load dialog
  • Update your current spritesheet if it has changed on disk with the  Update button
Because SpriteBreaker makes use of the latest web standards, people using older browsers may run into problems. Then again, not sure why anyone with a crappy browser would even be interested in this.
