About

SpriteBreaker is an open source web based sprite management tool build by Wilfred Godfrey. SpriteBreaker aims to make managing and updating spritesheets as simple as possible.

Quick start guide

  1. Load an image using the file selector
  2. Add and name a new sprite eg "mario"
  3. Add and name a new action eg "stand"
  4. Add a new frame
  5. Set frame coordinates
  6. Repeat
  • Scroll and click on the small canvas to set the view area
  • Left click on the large canvas to set the top left frame coordinate
  • Right click on the large canvas to set the bottom right frame coordinate
  • Scroll on the large canvas to zoom in and out
  • You can save and load using the buttons at the bottom middle.
